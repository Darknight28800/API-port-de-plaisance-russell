# Port de plaisance Russell

API Node.js / Express pour la gestion des catways et des réservations.

## Fonctionnalités

- Authentification utilisateur
- Gestion des catways
- Gestion des réservations
- Gestion des utilisateurs
- Dashboard
- Documentation API

## Installation

npm install

## Lancer le projet

node app.js

## Accès

http://localhost:8000
